# ZQX-ONE

Zaxxon port for the TI-99/4A home computer (1981).

February 2021.

## Requirements

TI-99/4A with:
* 32K RAM expansion
* F18A video processor enhancement (2012)
* Joystick

## Media

128 KiB ROM cartridge only. Cartridge image compatible with FinalGROM 99 and red boards.
